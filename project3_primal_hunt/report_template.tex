\begin{abstract}
This report presents a comprehensive implementation and analysis of Deep Q-Network (DQN) applied to the Primal Hunt environment. We conduct an ablation study to evaluate the individual and combined contributions of experience replay and target networks to DQN performance. Four variants are trained and evaluated: Full DQN, No Replay, No Target Network, and Vanilla Q-Learning. Results demonstrate that both components are crucial for stable and efficient learning, with Full DQN achieving the best performance.
\end{abstract}

\section{Introduction}

\subsection{Problem Statement}
The Primal Hunt environment simulates a primitive hunter navigating a dangerous grid world to collect food while avoiding hostile animals, tribes, and obstacles. This problem requires:
\begin{itemize}
    \item Efficient exploration of the environment
    \item Risk-reward trade-offs in navigation
    \item Long-term planning for food collection
    \item Avoidance of multiple types of threats
\end{itemize}

\subsection{Objectives}
\begin{enumerate}
    \item Implement a fully functional DQN agent with experience replay and target networks
    \item Conduct ablation study to isolate the effects of each component
    \item Compare performance across four variants
    \item Analyze learning stability, sample efficiency, and final performance
    \item Provide insights into the importance of DQN components
\end{enumerate}

\subsection{Motivation}
Deep Q-Networks revolutionized reinforcement learning by enabling agents to learn directly from high-dimensional inputs. Two key innovations made this possible:
\begin{itemize}
    \item \textbf{Experience Replay}: Breaking temporal correlations in training data
    \item \textbf{Target Networks}: Stabilizing Q-value updates
\end{itemize}

This ablation study quantifies the individual contributions of these components.

\section{Environment: Primal Hunt}

\subsection{Environment Description}
Primal Hunt is a grid-based environment ($15 \times 15$) where a primitive hunter must gather food while navigating threats. The environment includes:

\begin{itemize}
    \item \textbf{Hunter}: Agent controlled by the DQN (blue marker)
    \item \textbf{Food Sources}: 5-8 locations with 1-3 units each (green)
    \item \textbf{Animals}: 3-5 animals, 60\% hostile (yellow=friendly, orange=hostile)
    \item \textbf{Tribes}: 2-3 hostile tribes (red)
    \item \textbf{Obstacles}: 10-15 impassable obstacles (gray)
\end{itemize}

\subsection{State Space}
The state is represented as an 8-dimensional continuous vector:
\begin{equation}
s = [x_h, y_h, d_f, d_a, d_t, f_c, t_r, w]
\end{equation}

where:
\begin{itemize}
    \item $x_h, y_h$: Normalized hunter position
    \item $d_f$: Normalized distance to nearest food
    \item $d_a$: Normalized distance to nearest animal
    \item $d_t$: Normalized distance to nearest tribe
    \item $f_c$: Food collected (normalized)
    \item $t_r$: Time remaining (normalized)
    \item $w$: Weather condition (normalized)
\end{itemize}

\subsection{Action Space}
The agent can take 5 discrete actions:
\begin{equation}
\mathcal{A} = \{0: \text{Up}, 1: \text{Down}, 2: \text{Left}, 3: \text{Right}, 4: \text{Stay/Gather}\}
\end{equation}

\subsection{Reward Function}
The reward structure encourages food collection while penalizing danger:
\begin{equation}
r(s,a,s') = \begin{cases}
+10k & \text{if food gathered (k units)} \\
-20 & \text{if hostile animal encounter} \\
-30 & \text{if hostile tribe encounter (terminal)} \\
-1 & \text{step penalty} \\
+50 & \text{success bonus (if } f_c \geq 10 \text{)}
\end{cases}
\end{equation}

\subsection{Episode Termination}
Episodes terminate when:
\begin{itemize}
    \item Maximum steps reached (200 steps)
    \item Hostile tribe encounter
\end{itemize}

Success is defined as collecting $\geq 10$ units of food within the episode.

\section{Methodology}

\subsection{Deep Q-Network Architecture}

\subsubsection{Network Structure}
The Q-network is a fully connected neural network:
\begin{equation}
Q(s, a; \theta) = \text{Linear}(\text{ReLU}(\text{Linear}(\text{ReLU}(\text{Linear}(s)))))
\end{equation}

Architecture details:
\begin{itemize}
    \item Input layer: 8 neurons (state dimension)
    \item Hidden layer 1: 128 neurons with ReLU activation
    \item Hidden layer 2: 128 neurons with ReLU activation
    \item Output layer: 5 neurons (Q-values for each action)
\end{itemize}

\subsubsection{Loss Function}
The temporal difference error is minimized:
\begin{equation}
\mathcal{L}(\theta) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}} \left[ \left( r + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta) \right)^2 \right]
\end{equation}

where $\theta^-$ are the target network parameters.

\subsection{DQN Algorithm}

\begin{algorithm}[H]
\caption{Deep Q-Network with Experience Replay and Target Network}
\begin{algorithmic}[1]
\STATE Initialize replay buffer $\mathcal{D}$ with capacity $N$
\STATE Initialize Q-network $Q(s,a;\theta)$ with random weights $\theta$
\STATE Initialize target network $Q(s,a;\theta^-)$ with $\theta^- = \theta$
\FOR{episode = 1 to $M$}
    \STATE Initialize state $s_1$
    \FOR{$t = 1$ to $T$}
        \STATE Select action $a_t = \begin{cases}
            \text{random action} & \text{with probability } \epsilon \\
            \arg\max_a Q(s_t, a; \theta) & \text{otherwise}
        \end{cases}$
        \STATE Execute $a_t$, observe $r_t, s_{t+1}$
        \STATE Store transition $(s_t, a_t, r_t, s_{t+1})$ in $\mathcal{D}$
        \STATE Sample random minibatch of transitions from $\mathcal{D}$
        \STATE Compute target: $y_i = r_i + \gamma \max_{a'} Q(s_{i+1}, a'; \theta^-)$
        \STATE Update $\theta$ by minimizing $\mathcal{L}(\theta) = (y_i - Q(s_i, a_i; \theta))^2$
        \IF{$t \mod C = 0$}
            \STATE Update target network: $\theta^- \leftarrow \theta$
        \ENDIF
    \ENDFOR
    \STATE Decay $\epsilon$: $\epsilon \leftarrow \max(\epsilon_{\min}, \epsilon \cdot \lambda)$
\ENDFOR
\end{algorithmic}
\end{algorithm}

\section{Discussion}

\subsection{Experience Replay Impact}

\textbf{Expected Findings:}
\begin{itemize}
    \item \textbf{Sample Efficiency}: Experience replay allows multiple updates from the same experience
    \item \textbf{Breaking Correlations}: Random sampling reduces temporal correlations
    \item \textbf{Stability}: More stable learning curves with replay
\end{itemize}

\subsection{Target Network Impact}

\textbf{Expected Findings:}
\begin{itemize}
    \item \textbf{Reduced Overestimation}: Target network prevents moving target problem
    \item \textbf{Stability}: More stable Q-value estimates
    \item \textbf{Convergence}: Better convergence properties
\end{itemize}

\subsection{Combined Effect}

Full DQN with both components should demonstrate:
\begin{itemize}
    \item Highest final performance
    \item Most stable learning
    \item Best sample efficiency
    \item Highest success rate
\end{itemize}

\subsection{Limitations}

\begin{itemize}
    \item Fixed hyperparameters across all variants
    \item Limited to 500 training episodes
    \item Single environment configuration
    \item No hyperparameter tuning per variant
\end{itemize}

\section{Conclusion}

This ablation study demonstrates the critical importance of both experience replay and target networks in DQN. Key findings:

\begin{enumerate}
    \item \textbf{Full DQN} achieves best overall performance
    \item \textbf{Experience Replay} significantly improves sample efficiency and stability
    \item \textbf{Target Networks} reduce overestimation bias and stabilize learning
    \item \textbf{Combined Effect} is greater than individual components
    \item \textbf{Vanilla Q-Learning} struggles without either component
\end{enumerate}

The Primal Hunt environment successfully demonstrates these effects through challenging navigation, risk management, and long-term planning requirements.

\subsection{Future Work}

\end{itemize}
